# ⌨️ Type or Die

Zombies approach with words above their heads. Type to kill. Words escalate from "cat" to "pneumonoultramicroscopicsilicovolcanoconiosis".

▶️ [Play Now](https://ashleywolf.github.io/type-or-die/)

## How to Play

Open `index.html` in Chrome. Zombies shamble toward you with words floating above their heads. Type the word correctly to kill the zombie before it reaches you. You get 3 lives and 10+ waves of increasingly ridiculous vocabulary.

## Tech

Pure HTML5 Canvas and vanilla JavaScript. No dependencies. Single file, zero build step.

## Input

Keyboard (`KeyboardEvent`).
